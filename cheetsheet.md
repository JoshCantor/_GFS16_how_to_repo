# Git Cheatsheet

### Start a new project

```shell
$ mdkir project_name
$ git init
$ touch readme.md
$ git add readme.md
$ git commit -m "message"
```

### Do some work and then save it

First, do some work in a file that's already in the repo!

```shell
$ git status
$ git add <whatever_file>
$ git status
$ git commit -m"I made some changes"
```
